# Auralis Mock Player (HTML)

This repository contains only the standalone mock player runtime:

- `Auralis_mock_zenith.html`
- `auralis-core.js`
- `minutemen_library.js`

## Run

Open `Auralis_mock_zenith.html` in a modern Chromium browser.

For folder access features (library scan), use HTTPS or localhost where the File System Access API is supported.
