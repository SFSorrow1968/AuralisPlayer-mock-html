# Stylesheet Shards

The HTML shell links these files directly in filename order. Do not reorder links casually because later shards intentionally override earlier foundations.

- `00-foundation.css`: design tokens, reset, shell, generic layout, search/sidebar/media/list foundations.
- `01-player.css`: bottom nav, mini player, full player overlay, transport controls, inline queue shell.
- `02-controls-setup.css`: buttons, toggles, inline queue controls, setup/settings, sheets, home editor.
- `03-album-artist.css`: album detail, artist detail, onboarding, first-time setup, media setup.
- `04-zenith-overrides.css`: final Zenith UI overrides, dense rows/cards, title motion, interaction polish.
- `05-design-polish.css`: annotation-driven polish and targeted visual fixes.
- `06-shared-ui.css`: reusable control states and collection layout styles.
